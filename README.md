# Jardim Orbital AR

Experimento de Realidade Aumentada feito com A-Frame e AR.js para navegador.

## Como testar

1. Abra a aplicação em um navegador com câmera.
2. Permita o acesso à câmera.
3. Aponte para o marcador Hiro: <https://ar-js-org.github.io/AR.js/data/images/hiro.png>
4. Toque no núcleo 3D ou no botão "Evoluir" para alternar cores, movimento e som.

## Requisitos atendidos

- Realidade aumentada com AR.js.
- Elementos 3D com A-Frame.
- Interação por toque/click no núcleo e por botão na interface.
- Experiência multimídia com animação visual e som gerado no navegador.
